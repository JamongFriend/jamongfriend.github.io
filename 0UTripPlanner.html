0U Trip Planner

기술 스택: Node.js, Express, MySQL, React.js, Kakao API

프로젝트 개요

여행 계획을 작성하고 유저들끼리 공유할 수 있는 웹 플랫폼입니다. 프론트엔드는 React로 구현하고, 백엔드는 Node.js와 Express를 기반으로 RESTful API를 설계하였습니다.

주요 기능

  • 사용자 회원가입 및 로그인 (세션 기반)
  • 여행 계획 생성, 조회, 수정, 삭제
  • Kakao 소셜 로그인 및 지도 기반 장소 검색
  • CORS 설정을 통한 프론트 연동 보안 처리
  • Android Studio를 통해 모바일 버전도 연동

기술적 도전 및 해결

  • async/await을 활용한 비동기 처리 안정화
  • 쿠키 + 세션 미들웨어로 보안성 있는 인증 구현
  • 데이터베이스 설계 및 ERD 구성, 여행 일정 테이블 구조화
  • 클라이언트/서버 간 CORS 정책 대응
