Chatzar (실시간 보안 채팅 플랫폼)

⚙️ Tech: Java 17, Spring Boot 3.x, Spring Security, Spring Data JPA, MySQL, JUnit5, Mockito

프로젝트 개요

Chatzar는 보안성과 사용자 식별성을 극대화한 실시간 채팅 서비스의 백엔드 시스템입니다. 단순한 아이디 체계의 한계를 극복하기 위해 닉네임#태그(Discord 방식) 시스템을 도입하였으며, Spring Security와 JWT를 결합하여 견고한 인증/인가 아키텍처를 구축했습니다. 특히 테스트 주도 개발(TDD)과 테스트 픽스처 패턴을 활용하여 코드의 신뢰성을 확보하는 데 집중했습니다.

주요 기능 (Key Features)

  • 고유 식별자 시스템 (Nickname#Tag): 닉네임 중복을 허용하되 4자리 랜덤 태그를 부여하여 사용자를 고유하게 식별. findByNicknameAndTag 쿼리 최적화.
  • 보안 및 인증 아키텍처: BCrypt 기반 패스워드 암호화 및 JWT Refresh Token 재발급 로직 구현. 토큰 탈취 대비를 위한 Revoke(폐기) 시스템 적용.
  • 객체 지향적 친구 관계 설계: 회원 간의 다대다(N:M) 관계를 Friendship 중간 엔티티로 풀어내어 정규화된 데이터 모델 구축 및 친구 요청/수락 상태 관리.

트러블슈팅 (Troubleshooting)

  • 테스트 환경 데이터 정합성 이슈: BCrypt 암호화 규격 미달로 인한 로그인 테스트 실패 ➡️ BCryptPasswordEncoder를 내장한 정적 테스트 픽스처(MemberFixture) 도입으로 해결.
  • 검색 성능 및 사용자 편의성: 단순 닉네임 검색 시 중복 사용자 과다 노출 ➡️ 닉네임과 태그를 조합한 복합 인덱스(Composite Index) 활용 및 정확한 1인 타겟팅 검색 로직 구현.
  • Refresh Token 무한 재사용 방지: 이미 사용된 리프레시 토큰의 보안 취약점 ➡️ DB 레벨에서 isUsed 상태 체크 및 예외 처리를 통해 토큰 재발급 로직 강화.
