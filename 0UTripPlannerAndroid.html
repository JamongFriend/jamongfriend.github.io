0U Trip Planner (Android Ver.)

기술 스택: Android Studio, Java, Node.js (Express)

프로젝트 개요

0U Trip Planner의 Android 기반 확장 버전입니다. 기존 Node.js 백엔드 API를 재사용하여 모바일 앱 환경에서 여행 계획 생성, 조회, 수정 등의 기능을 제공합니다.

주요 기능

  • 여행 계획 조회 및 상세 화면 UI 구성
  • 회원가입, 로그인, 로그아웃 기능 구현
  • REST API와의 HTTP 통신을 통한 데이터 연동
  • CORS 문제 해결 및 보안 설정 적용

기술적 도전 및 해결

  • 백엔드 API와의 통신을 위한 HTTP 클라이언트 구성 (Retrofit 등)
  • 모바일 환경의 CORS 정책 문제 해결
  • 화면 전환 및 데이터 바인딩 처리
