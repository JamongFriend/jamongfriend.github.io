Soul of Herb Seller

기술 스택: Unreal Engine 5, Blueprint

프로젝트 개요

식물을 채집하고 보스 몬스터를 사냥하는 액션 어드벤처 게임으로, 블루프린트를 활용하여 게임 로직을 설계하고 구현하였습니다.

주요 기능

  • 미니게임 기반의 식물 채집 시스템 구현
  • 보스 몬스터 등장 및 전투 시스템 설계
  • 상태 저장, 카메라 무빙, HUD 연동
  • 키보드 입력 기반 인벤토리 토글 UI 시스템 구현

객체지향 설계 요소

  • 플레이어 캐릭터와 몬스터 간 상호작용 설계
  • 이벤트 중심의 상태 전이와 게임 흐름 구성
  • BP_Player / BP_QD_Player 구성 간 Pawn 관리 이슈 해결

기술적 도전 및 해결

  • 맵 전환 시 상태 유지 문제 해결 (세이브/로드 적용)
  • 이벤트 BeginPlay 시 중복 실행 문제 디버깅
  • 미니게임 성공 후 초기 위치로 되돌아가는 이슈 해결
