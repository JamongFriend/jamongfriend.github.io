Unmanned Order Machine

기술 스택: Java

프로젝트 개요

Java 언어 기반의 콘솔 키오스크 프로그램으로, 사용자로부터 입력을 받아 음식을 주문하고 영수증을 출력하는 시스템입니다. 객체지향 설계 원칙을 실제 코드로 구현하며 학습한 프로젝트입니다.

주요 기능

  • Rice/Noodle/Drink 메뉴 선택 및 수량 입력
  • 메뉴에 따른 Food 객체 생성 및 주문 처리
  • 총 주문 금액 계산 및 영수증 출력 기능

객체지향 설계 적용

  • Food 추상 클래스 + 상속 구조를 통한 다형성
  • Menu, Order, MenuManager 클래스로 기능 분리
  • 단일 책임 원칙(SRP) 및 결합도 최소화를 위한 설계 구조
  • 추가 메뉴/기능 확장을 고려한 유연한 구조 설계
